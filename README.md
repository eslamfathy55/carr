# EgyCar - Premium Luxury Car Website

## 🚀 Overview

EgyCar is a modern, premium luxury car website built with **React** and **Tailwind CSS**. It showcases luxury vehicles, services, and provides an elegant user experience.

## ✨ Features

- 🎨 Modern dark theme with red accents
- 📱 Fully responsive design
- 🚗 Luxury car showcase
- 💼 Professional services section
- ⭐ Customer testimonials
- 📞 Contact form with social media links
- 🎯 Smooth scrolling navigation
- 🏆 Stats and achievements display

## 🛠️ Tech Stack

- **React 18.2.0** - UI Library
- **Tailwind CSS 3.3.0** - Styling
- **PostCSS** - CSS Processing
- **Vite** - Build tool (recommended)

## 📦 Installation

1. Clone the repository:
```bash
git clone https://github.com/eslamfathy55/carr.git
cd carr
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
